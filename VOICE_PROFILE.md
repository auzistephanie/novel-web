# 顧事 Voice Profile — House Style v1

> 2026-07-10 由 Fable session 經 STEP 0 試讀鎖定（A+C+D 三聲道，B 落選）。
> 用法：生成內容時整段塞入 prompt，按下方分配表揀 voice。

## 通用 House Rules（三聲道共用）

- 全繁體中文書面語，用字自然無翻譯腔；對白可帶口語感但唔用方言字
- 視角：第三人稱貼身限知（跟主角行）；Voice C 例外用第一人稱
- 段落短：1–3 句一段，手機閱讀友好；關鍵句獨立成段
- **開場鈎子公式（頭 3 句）**：① 精確場景錨點（具體時間／動作）② 一件事唔對路 ③ 主角反應同讀者預期相反
- 爽點／情感點密度：每 300–400 字一個小爆點（打臉／反轉／揭底／金句）
- 金句：每篇 1–2 句，放喺轉折位或末句，唔准堆砌

## 收尾公式

- **serial cliffhanger**：最後一段「新資訊 + 未完成動作」同時出現，斬喺動作中間；最尾一句 ≤15 字
- **short 情感收筆**：90% 篇幅鋪日常（可微搞笑）→ 一個具體物件揭底 → 淚點／會心一句突襲收筆，唔畀讀者心理準備；末句要可以獨立引用

## Voice A —「快刀流」

**適用**：serial 男頻（重生、系統流、玄幻、末世、無限流、都市異能戰鬥向）

- 句子短過刀：平均句長 ≤15 字，動詞開頭多
- 三句一小爽；開場 100 字內主角必須拎到「牌」（金手指／轉機）
- 對白:敘述 ≈ 3:7，對白必須推進或打臉，唔准寒暄
- 情緒底色：熱血＋憋屈翻身——「被踩三年，輪到我」係核心爽感

## Voice C —「夜行者」

**適用**：都市搞笑、輕鬆向、社畜題材（serial / short 皆可）

- 第一人稱毒舌，自嘲先行：主角永遠先損自己，再損世界
- 冷幽默公式：異常事件 × 打工人日常類比（「全店定格＝被老闆抽走靈魂的我」）
- 遇大事反應淡定得離譜，用小動作（掃碼、付款）製造反差笑點
- 每 2–3 段一個抖包袱位；金句多為「歪理總結」

## Voice D —「拾光人」（突襲派）

**適用**：short 情感篇（暗戀、愛情、親情、家庭倫理、甜寵）

- 結構：90% 日常輕鬆 → 具體物件揭底 → 一句突襲淚點，「好突然」係靈魂
- 嚴守短篇情感四硬規：單一主導情感／具體場景對白物件推進／明確淚點收筆／情緒有轉折（輕→重）
- 淚點載體必須係「日常被忽略嘅細節」（永遠剩最後一份嘅蘿蔔）
- 唔准直寫「他很感動／她哭了很久」——用動作寫情緒（「轉身出門，眼淚才敢掉下來」）

## 用詞禁忌

- 禁：翻譯腔（「哦我的天」）、說教、無鋪墊開掛、爛尾、性描寫
- 禁概括式情緒詞做主力：感動、心痛、震驚（一律用場景／動作代替）
- 禁 AI 腔陳套開場：「夜幕降臨」「不禁」「彷彿整個世界」

## 三個簽名特徵（一睇就知顧事出品）

1. **精確時間錨點開場**（「十一點零七分」式）
2. **主角反應同讀者預期相反**（大事淡定、小事上心）
3. **末句短、可獨立引用**，斬喺情緒最高點
